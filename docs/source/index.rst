Numpy Katas
===================
`numpy-katas`_ is a growing collection of machine learning and probabilistic models and algorithms
written exclusively in `NumPy`_ and the Python `standard library`_.

The purpose of the project is to provide reference implementations of common
machine learning components for sharping your ML skills and cracking interviews.

.. _numpy-katas: TBU
.. _NumPy: https://numpy.org/
.. _standard library: https://docs.python.org/3/library/
